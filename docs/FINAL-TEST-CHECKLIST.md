# Final Test Checklist — Fictional Data Only

Complete these checks after deploying version 1.0.0.

## Access and installation

- [ ] Approved Microsoft account opens the dashboard.
- [ ] An unauthorised account is rejected.
- [ ] Role shows Administrator.
- [ ] Version shows 1.0.0.
- [ ] Install app works, or Edge provides the manual Apps installation option.
- [ ] Installed PWA opens and signs in correctly.

## People and safety

- [ ] Create a fictional family.
- [ ] Add a fictional guardian.
- [ ] Create two fictional sibling students.
- [ ] Link guardians to the students.
- [ ] Add a fictional emergency contact.
- [ ] Add a fictional medical note and safety alert.
- [ ] Safety alert appears on Attendance.
- [ ] Archive and restore an eligible fictional record.

## Enquiries and training

- [ ] Create an enquiry.
- [ ] Create and complete a follow-up.
- [ ] Convert an enquiry to a student.
- [ ] Create a term.
- [ ] Generate Tuesday and Thursday sessions.
- [ ] Edit and cancel a test session.
- [ ] Save attendance for a session.

## Development

- [ ] Create a grading event.
- [ ] Record a passed grading and update the belt.
- [ ] Create a progress review.
- [ ] Create and complete a student goal.

## Finance

- [ ] Create the approved fee schedule.
- [ ] Create a $120 first-student charge.
- [ ] Create a $100 sibling charge.
- [ ] Record one $150 family payment.
- [ ] Allocate $120 and $30.
- [ ] Confirm $70 remains outstanding.
- [ ] Print a receipt.
- [ ] Create and print an invoice.
- [ ] Reverse a fictional payment and verify balances update.
- [ ] Create an expense and expense payment.
- [ ] Create a recurring expense occurrence.

## Banking and reports

- [ ] Import the sample or a fictional Kiwibank CSV.
- [ ] Confirm, reject and match fictional transactions.
- [ ] Save a fictional reconciliation.
- [ ] Run each report.
- [ ] Export a report to CSV.

## Backup and audit

- [ ] Create an encrypted backup.
- [ ] Confirm the file cannot be opened with the wrong password.
- [ ] Validate it with the correct password.
- [ ] Review backup history.
- [ ] Review audit events.
- [ ] Test merge restore only in a safe fictional-data environment.

Do not enter real medical or banking information until the relevant checks pass.
